打字游戏

发表于 2017-04-16 | 分类于 网页制作 | |



一个真正的网页项目从来不是一个HTML文件所能表示的（如百度主页实战笔记、3d轮播图那样），而是有许许多多的模块组成。
软件设计方面的最高要求————高内聚低耦合，同样也适用于网页。
高内聚低耦合是判断设计好坏的标准，主要是面向对象的设计，主要看类的内聚性、耦合度。
内聚性：块内联系，模块内部各元素联系的紧密度越高内聚性越高。
耦合度：块间联系，各个模块之间联系的紧密度越低其耦合度越低。
本次的打字游戏我们将其分成Html、Js、Css三个部分。在HTML中以link引用Css文件，以script引用Js。
